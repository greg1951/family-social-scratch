Background
Currently what is working correctly in Family Threads home page is a private message does not allow the subject and title to be seen by anyone other than the sender and recipient, and only when they open it. 
  - In the list of messages the Message column for that thread reads "Private Thread message hidden". 
  - The Visibility column contains "Private"

Request
What is not working correctly in Family Threads, is when a new thread is being created: 
- A "private" visibility allows the author to select one or more recipients.
- While a "public" visibility does not allow the author to select specific recipients, and instead sends the message to all family members.

Regardless of whether it is private or public, always shows the first name of the recipients in the Recipients column on the Family Threads home page.
Private versus Public visibility only means that the subject and title of the message are not shown on the Family Threads home page.

Other changes to the Family Threads home page include:
- The Type column should be removed.
- In it's place should be a replies icon with a count of the number of replies in that thread.
- If there are images attached to the thread, then there should be an image icon with a count of the number of images attached to that thread.
