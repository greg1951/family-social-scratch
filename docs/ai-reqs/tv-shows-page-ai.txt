
Create a home page in the (features)/(tv) router group that consists of three sections. In the first section, entitled "Latest tv shows" create a horizontally scrollable (use arrow icons for left and right) set of four tv show images, each with the name of the show and a date. If it is a small screen then the scrolling should be vertical. 

There are sample images to be found at this location on the file system: 
C:\Users\ghughlett\Projects\my-projects\family-social-prototype-images\photos\tv-shows

In the second section of the page, create a another scrollable list of four pictures for a section entitled "Top Rated TV Shows". Each of the pictures should have a caption consisting of the show name, a five-star rating icons and a thumbs up icons with the number of recommendations. 

The third section of the page will be a section entitled "Show Finder" with a search bar. Below the Show Finder form will be a list of selectable TV shows. The columns in the TV Shows list are: Show Name, Thumbs Up number of recommendations, Average Rating, # of Seasons, Genre, Channel, Added By (member first name) and the number of comments on the show. The first and second sections will occupy half the page and the third section the remainder.

The layout.tsx for this page should reuse the root app folder layout.tsx.
